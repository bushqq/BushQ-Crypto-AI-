# Crypto Intelligence Center 使用说明

## 最简单用法

双击：

```text
启动CIC.bat
```

然后在菜单里选择：

- `1` 启动自动推送软件
- `2` 立即生成并推送一次报告
- `3` 检查数据源健康状态
- `4` 打开报告目录
- `5` 编辑 AI Prompt
- `6` 安装/更新依赖

## 直接运行

立即生成并推送一次：

```powershell
.\立即运行一次.bat
```

常驻自动推送：

```powershell
.\自动推送运行.bat
```

健康检查：

```powershell
.\健康检查.bat
```

## 文件位置

报告目录：

```text
data\reports
```

AI Prompt：

```text
templates\prompts\daily_analysis.md
```

密钥配置：

```text
.env
```

## 注意

`.env` 保存了 API Key 和企业微信机器人地址，不要发给别人。
